# 🌟 DreamClass  
**Plataforma web para docentes particulares**

[![PHP](https://img.shields.io/badge/PHP-7.4%2B-777BB4?logo=php&logoColor=white)](https://www.php.net/)
[![MySQL](https://img.shields.io/badge/MySQL-8.0%2B-4479A1?logo=mysql&logoColor=white)](https://www.mysql.com/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

DreamClass es una aplicación web diseñada para **simplificar la gestión diaria de docentes particulares**. Permite reservar clases, comunicarse con alumnos, asignar tareas creativas e interactuar de forma personalizada, todo desde una única plataforma **gratuita, segura y responsive**.

> 🎯 *"Donde la enseñanza se encuentra con el sueño."*

---

## ✨ Funcionalidades

- ✅ **Registro e inicio de sesión seguro**
- 🗓️ **Calendario interactivo** para reservas de clase
- 💬 **Chat privado** entre docente y alumno
- 📝 **Tareas "inspiradas en sueños"** para motivar el aprendizaje
- ⭐ **Feedback post-clase** con calificación y comentarios
- 🔔 **Notificaciones automáticas por email** (registro, reserva, tarea)
- 📱 **Diseño 100% responsive** (móvil, tablet, PC)
- 🔒 **Validaciones frontend y backend** para seguridad
